# SauceHub — Issue Tracker

This repository is used exclusively for tracking bugs, feature requests, and general feedback for [SauceHub](https://saucehub.app) — a community-driven platform for discovering, reviewing, and discussing hot sauces.

There is no source code here. For questions or discussion, use the [Forum](https://saucehub.app/forum) on the site itself.

## Reporting a Bug

Before opening an issue, please search existing issues to avoid duplicates.

When reporting a bug, include:

- A clear description of what went wrong
- Steps to reproduce it
- What you expected to happen vs. what actually happened
- Your browser and operating system (if relevant)
- Screenshots or screen recordings if helpful

## Requesting a Feature

Feature requests are welcome. Describe the problem you're trying to solve and how you'd expect the feature to work. The more context, the better.

## Scope

This tracker covers:

- The SauceHub web app (saucehub.app)
- Account and authentication issues
- Sauce catalog, submissions, and reviews
- Lists, collections, and recipes
- Forum functionality
- General UX and accessibility

It does **not** cover:

- Third-party login providers (Google OAuth)
- Email delivery problems outside our control

## Labels

| Label | Meaning |
|---|---|
| `bug` | Something isn't working correctly |
| `enhancement` | New functionality request |
| `question` | Needs clarification before triaging |
| `duplicate` | Already tracked elsewhere |
| `wontfix` | Out of scope or intentional behavior |
| `invalid` | Not a valid bug report or feature request |

Thanks for helping make SauceHub better.
